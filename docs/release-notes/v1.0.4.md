# D35E Piecemeal Armor And Called Shots v1.0.4

This release adds local armor AC support for called shots in D35E's native Apply Damage workflow.

## New

- Added `Called-shot local armor AC`, a world setting with three modes:
  - `Adjust AC in Apply Damage`;
  - `Show adjustment only`;
  - `Disabled`.
- Called shots now carry their selected location into D35E Apply Damage buttons.
- When enabled, D35E's AC Details can show entries such as `Called Shot Local Armor: Leg -1` before hit and crit resolution.

## Integration

- The module still does not edit D35E system files.
- Attack penalties continue to flow through `ChatAttack.addAttack` and D35E's native modifier breakdown.
- Local armor AC uses `D35E.DamageRoll.preHitCheck`, after D35E calculates defense AC and before D35E decides hit or crit.

## Compatibility

- Foundry VTT: verified on 14.362.
- D35E system: verified on 3.0.2.

## Rules Boundary

The bundled called-shot and piecemeal armor defaults are PF1e-derived optional-rule scaffolding. They are configurable and are not represented as official D&D 3.5 RAW.
