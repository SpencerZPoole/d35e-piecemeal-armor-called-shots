# D35E Piecemeal Armor And Called Shots v1.0.5

This patch fixes the live Apply Damage path for called-shot local armor AC.

## Fixed

- Native D35E damage cards now retain the selected called-shot location, so clicking Apply Damage can adjust AC with `Called Shot Local Armor: <Location>`.
- Generated aggregate armor rows no longer show the piecemeal-component configure icon.
- Generated aggregate armor item sheets no longer show the component configuration panel.

## Compatibility

- Foundry VTT: verified on 14.362.
- D35E system: verified on 3.0.2.
