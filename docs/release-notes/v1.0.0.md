# D35E Piecemeal Armor And Called Shots v1.0.0

This is the first public release of the module.

## Highlights

- Piecemeal armor components can be synced into one D35E-native aggregate armor item.
- Armor component data can be restored if a GM wants to reverse a sync.
- Called-shot selection now lives inside D35E's native attack/use dialog.
- Called-shot penalties are injected into D35E attack math and appear in the expanded attack breakdown.
- Full attacks can ask per attack, apply to the first attack only, apply to every attack, or ignore called-shot selections.
- GM-confirmed chat cards apply configurable normal, critical, or debilitating outcomes.
- The profile editor lets GMs customize locations, penalties, coverage slots, and outcome effects.

## Compatibility

- Foundry VTT v14.362 verified.
- D35E 3.0.2 verified.
- Foundry v13 is not marked verified yet.

## Install

Use this manifest URL in Foundry:

```text
https://github.com/SpencerZPoole/d35e-piecemeal-armor-called-shots/releases/latest/download/module.json
```

## Caveat

The bundled called-shot and piecemeal armor defaults are PF1e-derived optional-rule scaffolding. They are configurable and are not represented as official D&D 3.5 RAW.
