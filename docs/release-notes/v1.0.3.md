# D35E Piecemeal Armor And Called Shots v1.0.3

This packaging release updates the public release surfaces and replaces mock package screenshots with live Foundry screenshots.

## Public Release Updates

- Added a Foundry Package Release API publisher for the GitHub release workflow.
- Updated README, package-listing notes, and Foundry package description version text to the current release.
- Replaced public screenshots with captures from a live Foundry VTT 14.362 D35E 3.0.2 session.
- Tightened validation so stale public version references are caught before release.

## Compatibility

- Foundry VTT: verified on 14.362.
- D35E system: verified on 3.0.2.

## Rules Boundary

The bundled called-shot and piecemeal armor defaults are PF1e-derived optional-rule scaffolding. They are configurable and are not represented as official D&D 3.5 RAW.
